                              DOSYA AÇIKLAMALARI
================================================================================

BACKEND (Flask - Python)
------------------------

app.py
  - Flask web sunucusu
  - API endpoint'leri
  - Yüz tanıma işlemleri
  - Veri yönetimi
  
  API Endpoints:
  • POST /api/register          - Öğrenci kayıt
  • POST /api/attendance         - Yoklama işaretle
  • GET  /api/students           - Öğrencileri listele
  • GET  /api/attendance/records - Yoklama kayıtları
  • DELETE /api/students/:id     - Öğrenci sil

templates/index.html
  - Basit web arayüzü
  - Kamera entegrasyonu
  - JavaScript ile API çağrıları

WEB PANELİ (React)

================================================================================
                              VERİ AKIŞI
================================================================================

1. ÖĞRENCİ KAYIT AKIŞI
   -------------------
   Kullanıcı → Kamera → Fotoğraf Çek → Base64 Encode → 
   API POST /api/register → Yüz Tespiti → Dosya Kaydet → 
   JSON Güncelle → Başarı Mesajı

2. YOKLAMA AKIŞI
   -------------
   Kullanıcı → Kamera → Fotoğraf Çek → Base64 Encode → 
   API POST /api/attendance → Yüz Tanıma → Karşılaştır → 
   Eşleşme Bul → Kayıt Oluştur → JSON Güncelle → 
   Sonuç Göster

3. LİSTELEME AKIŞI
   ---------------
   Kullanıcı → API GET İsteği → JSON Oku → 
   Veri Dönüştür → UI Güncelle
